# V9内容
1、重要：增加了“GAMESTART”的小游戏网页项目，点击URL或者部署本地双击即可体验。（小游戏内容需要先单击渐变页进入渐变环，点击渐变环内部的选项即可（个人网页特色））    
2、修改了环，变成可点击互动有过渡动画的网页，不再那么食之无味。  

3、似乎更简洁了？  

GAMESTART小游戏内容为“Chinese-Chess”（中国棋）、“Barrage-Arrow”（弹幕箭）、“Retro-Snakes”（贪吃蛇）、“Glass-Beads”（玻璃珠）
玩法：
1.象棋：和经典的中国象棋一致，如果不会玩可以在游戏内右上角点击Q版人物（FQ：常见问答）即可查看象棋规则，实在不行点击象棋本体也可以看到明显的提示。

2.消除弹幕：消除贬义词使用“蓝箭”让它变为褒义词，“红箭”可以让碎掉の心变为爱心，“黄箭”可以让贬义词禁止不动3秒。武器切换就在右下角。

3.贪吃蛇：下方四个按钮为↑（上）↓（下）←（左）→（右），通过吃小红点让蛇身变长，碰墙可到另一边，碰到自己游戏结束。

4.小球逃离环状体：晃动手机让小球通过缝隙离开，离开即可获胜，当然你不动也会赢（仔细看环会稍微移动，参考星轨运动轨迹），需要等待漫长的时间。
